# Typed Parser

[![Build Status](https://travis-ci.org/jinjor/typed-parser.svg)](https://travis-ci.org/jinjor/typed-parser)

A parser library for TypeScript, inspired by elm/parser and Parsec(Haskell).

```shell
$ npm i typed-parser
```
